# **App Name**: CoirConnect

## Core Features:

- Product Catalog: Display and manage coir products with images, specs, and downloadable brochures. Implemented in Firestore.
- Tender Management: Dynamically list active tenders with key details and downloadable documents. Stored in Firestore.
- SEO Optimization: Implement meta tags, structured data (schema.org), and performance enhancements for faster loading.
- Contact Form with Inquiry Routing: A contact form for general and tender-specific inquiries. Integrates with Firebase for data management.
- Job Openings Display: Dynamically show job openings and application instructions.
- News Feed: Display latest news from KSCMMC
- AI-Powered FAQ: The website incorporates a tool that leverages generative AI to address user questions and concerns.

## Style Guidelines:

- Primary color: Deep teal (#008080) to reflect the eco-friendly nature of coir and Kerala's backwaters.
- Background color: Soft off-white (#F8F8FF), slightly tinted with the primary hue.
- Accent color: Earthy beige (#B8860B) to complement the natural coir and add a touch of warmth.
- Headline font: 'Inter' (sans-serif) for modern and clean headings.
- Body font: 'PT Sans' (sans-serif) for readable and accessible body text.
- Use line icons that are simple and nature-inspired, with subtle animations.
- Responsive design with a clean and minimalistic layout for optimal user experience on all devices.
- Smooth transitions and subtle animations on user interactions (e.g., light/dark mode toggle, page transitions).
